# 🛍️ E-commerce Customer Analytics Dashboard

An interactive Power BI dashboard designed to analyze e-commerce sales performance, customer purchasing behavior, revenue by product categories, and geographical order distribution. This project converts transactional retail data into high-impact visual stories to optimize revenue streams and target top customer segments.

---

## 📌 Project Overview
Understanding individual customer value and category velocity is essential for modern e-commerce expansion. This repository features an end-to-end business intelligence solution evaluating vital retail health metrics. 

The dashboard provides deep insights into total orders, overall volume quantities, net revenue benchmarks, and granular customer purchasing power tracking.

---

## 📂 Repository Structure
The files in this repository are organized as follows:

*   **`E-commerce Dashboard.pdf`**: A high-resolution static report containing the finalized layout of the sales dashboard.
*   **`E-commerce Task 6.pbix`**: The master Power BI desktop file holding data structures, model interactions, and custom canvas visuals.
*   **`E-commerce_Data.xlsx`**: The source e-commerce transactional database used for the pipeline.

## 🚀 Key Features & Insights
*   **Performance Overviews**: Immediate visibility into core operational figures like total orders (1K+), item quantities (3K+), and gross revenue ($657.16K).
*   **Top Customer Leaderboards**: Isolated sorting of top-tier consumer profiles mapping individual historical spending metrics.
*   **Category Performance & Slicers**: Interactive category distributions (e.g., Clothing, Sports) mapped cleanly alongside active fulfillment status filters (Canceled, Delivered, Pending).
*   **Global Distribution**: Geospatial order mapping tracking structural revenue density across global target hubs.

---

## 🛠️ Tools & Technologies Used
*   **Power BI Desktop**: Advanced canvas visual structuring, interactive cross-filtering, and dynamic KPI mapping.
*   **Power Query**: Structural text data corrections, handling data formats, and structural column organization.
*   **DAX (Data Analysis Expressions)**: Custom formulas optimizing financial metric sums and distinct tracking logic.
